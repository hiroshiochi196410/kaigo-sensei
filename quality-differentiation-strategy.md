## ✅ まとめ: 回数以外の差別化要素

1. **AI応答の長さ**: 30文字 → 60文字 → 100文字 → 180文字
2. **語彙の難易度**: N5 → N5-N4 → N4-N3 → N3-N2
3. **文法の複雑さ**: 基本形 → です・ます → 敬語 → 謙譲語
4. **シナリオ数**: 3種 → 5種 → 8種 → 12種
5. **フィードバックの質**: 励まし → 助言 → 分析 → AI診断
6. **表示言語**: 3段 → 3段 → 2段 → 1段
7. **例文保存**: なし → 50件 → 200件 → 無制限
8. **学習分析**: なし → 簡易 → 詳細 → AI分析
