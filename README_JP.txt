入浴カテゴリ：例文ローテーション完全版（ZIP）
========================================

目的:
  入浴（bath）の例文を「カテゴリ」で選べて、ボタンを押すたびに例文が切り替わるようにします。

ファイル:
  examples.json  … 入浴カテゴリ(5種) × persona(5種) × 例文(各5) = 125文
  index.html     … 動作確認用デモ（例文ローテーション実装済み）
  README_JP.txt  … この説明

反映手順（最短）:
  1) リポジトリ直下の examples.json を、このZIP内 examples.json に置き換え
  2) 動作確認したい場合、index.html もこのZIP版に一旦置き換え
  3) Vercel自動デプロイ後、入浴カテゴリを切り替えて「例文を入れる」を連打 → 例文が変わることを確認

本番UIへ移植する場合:
  - fetch("/examples.json") で読み込み
  - persona と bathCategory をキーにして配列から取り出し
  - counters[persona][cat] を持ち、押すたびに i を進める

次の拡張:
  - 入浴以外（食事/排泄/夜間/クレーム）も同様にカテゴリ設計へ展開可能
